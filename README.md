# qmapc
Map file parser and converter for the Quake FPS games


## Usage

   Not ready yet.
   

## Building

   This project uses Maven. To build:
   
   `mvn package`
   

## Tests

   Unit tests using junit are included. They should be run automatically during the build. To run them manually:

  `mvn test`
